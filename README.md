### 🌶️ Customy

A customer service shift management system.

### 📦 Installation

Python dependencies:

```
pip install -r requirements.txt
```

Node dependencies:

```
npm install
```

### ⚡️ Development

```
flask --app main run --debug
```

### 🎨 Tailwind

```
npx tailwindcss -i ./app/static/css/tailwind.css -o ./app/static/css/styles.css --watch
```
